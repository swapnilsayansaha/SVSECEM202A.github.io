Headpose Results:
Figure: Number of frames of all 27 targets for head movement from origin to target and target to origin (apart from these, there also exists target-target head movement).
For head-pose estimation, To prevent overfitting 3-fold cross validation and 90:10 holdout (train:test) were utilized. Initially, we tested performance of several classifiers via trial-and-error. Afterwards, once we found out the base classifier to work on (based on classification accuracy), grid-search hyperparameter optimization was used for searching for ideal hyperparameters for the chosen classifier. 37 spatial features were extracted from the dataset.

The hyperparameters include:
  • Eta: used for shrinking weights in each step similar to learning rate of Gradient Boosted models. Chosen range: [0.001, 0.01, 0.10]
  • Min_child_weight: this controls the minimum sum of weights and tree splitting. This helps in preventing overfitting and larger value is chosen to make the model more conservative. Chosen range: [1, 3, 5, 7]
  • Max_depth: This controls the depth of the tree, as the data is not too high, we restrict the depth to a small number and vary this to find optimal depth. Chosen range: [6, 15, 45]
  • Gamma: As the tree is of low depth, gamma the regularization factor will help in improving test accuracy. The value is varied over [0, 1, 5].
  • Colsample_bytree: this controls the number of features given to a tree and value varied between [0.5, 0.7, 0.8]
  • N_estimators: This decides the number of trees to fit.

For kalman filter, median delta motion errors of 8.3 and 5.6 degrees were observed in azimuth and elevation planes respectively. For machine learning approach, least mean absolute error (MAE) is 17.7 and 7.46 degrees and best classification accuracies of 91.93% and 96.88% in azimuth and elevation planes respectively were observed using XGBoost (grid search hyperparameter optimized and SMOTe-utilized). Obtained MAE is well within the human binaural sound localization range (+- 20 degrees).

Human Activity Results:

For activity classification, a sliding window of 1 second with stride length of 0.5 seconds was chosen. 241 spatial features were extracted from 8002 frames of 9 activities. Explored classifiers include SVM, bagged trees, boosted trees, decision trees and shallow neural networks. To prevent overfitting, holdout validation (ratio: 85:15) was used. The hyperparameters of the classifiers (initially chosen via trial-and-error) are as follows:
​
  • Bagged_45: Bagged decision trees; number of learners: 45; maximum number of splits: 8001.
  • Bagged_60: Bagged decision trees; number of learners: 60; maximum number of splits: 8001.
  • Adaboost_60: Boosted decision trees; number of learners: 45; maximum number of splits: 60; learning rate: 0.1; 
  • Adaboost_80: Boosted decision trees; number of learners: 45; maximum number of splits: 80; learning rate: 0.1.
  • QSVM: 1-vs-1 SVM with quadratic kernel; kernel scale: auto; box constraint level: 1; standardize data: true.
  • CSVM: 1-vs-1 SVM with cubic kernel; kernel scale: auto; box constraint level: 1; standardize data: true.
  • Coarse_DT: Coarse decision tree; maximum number of splits: 60; split criterion: maximum deviance reduction; surrogate decision splits: Find all.

Later, Bayesian optimization without any training time limit was applied to CSM and Ensemble, using expected improvement per second plus as acquisition function and 30 iterations. The hyperparameter search ranges are as follows:
  • Ensemble: Method: Bag, Adaboost, RUSBoost; number of learners: [10,500]; learning rate: [0.001-1]; maximum number of splits: [1-8001]; number of predictors to sample: [1-241].
  • SVM: Multiclass method: 1-vs-1 and 1-vs-all; box constraint level: [0.001,1000]; kernel scale: [0.001,1000]; kernel functions: Gausian, linear, quadratic, cubic; standardize data: true; false.


Bayesian optimization recommended using bagged decision trees with 377 learners, 11 predictors to sample and 7969 maximum number of splits and 1-vs-1 SVM with quadratic kernel, box constraint level of 1.0583 and data standardization set to true.

Two-layer feedforward neural networks with n sigmoid hidden neurons in a single hidden layer and softmax output, were trained using scaled conjugate gradient backpropagation and using cross-entropy as performance parameter (calculations: MEX) using 70:15:15 holdout ratio for training, testing and validation sets (data division: random) The only hyperparameter here is the number of hidden neurons.

Best supervised classification performance was obtained using CSVM, with a classification accuracy of 94.6% and SNN_15 (15 sigmoid neurons), with an test accuracy of 94.25%.

Binaural Audio Results:
A problem with HRTF datasets is that they are not individualized. Different people have distinct sound localization capabilities, but it is generally assumed that the localization range is within +- 20 degrees. This is confirmed via our real-time localization test, where a subject aims to point out where a particular sound source might be based on hearing.

We observed a median range of 40 degrees, which is +- 20 degrees of localization accuracy of humans. This complements the MAE of our classifier, which is much less than 40 degrees.
